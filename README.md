# react-native-deezer

## Getting started

`$ yarn i react-native-deezer`

or
`$ npm install react-native-deezer`

`$ react-native link react-native-deezer`


## Try it
```javascript
import Deezer from 'rn-deezer';

Deezer.getModuleName((msg) => console.log(msg))

```
