# react-native-deezer

## Getting started

`$ yarn i rn-deezer-sdk`

or

`$ npm install rn-deezer-sdk`

`$ react-native link rn-deezer-sdk`


## Try it
```javascript
import Deezer from 'rn-deezer';

Deezer.getModuleName((msg) => console.log(msg))

```
